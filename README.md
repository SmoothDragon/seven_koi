# Koi

Matching card game built on XOR / parity: **six breeds**, **32** cards, baseline tableau **8** face-up cards.

Locked roster: see [koi_selection.md](koi_selection.md). Rules: [rules/RULES.md](rules/RULES.md).

This repository’s **default product** is that base game. A separate optional **Koi** expansion (seven species) adds Kumonryu and a **64-card** odd-weight deck with **`L₀ = 10`**, including the all-seven card; it is documented only as a possible future SKU — see [`expansion/seven_koi/README.md`](expansion/seven_koi/README.md).
